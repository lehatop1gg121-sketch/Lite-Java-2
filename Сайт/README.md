# Архитектура проекта InCode (Платформа для изучения Java)

Этот файл содержит описание архитектуры сайта, специально адаптированной для школьного проекта. 
Все работает без Node.js или серверов напрямую в браузере.

## 📂 Структура папок

*   `index.html` — Главный файл с HTML каркасом. Именно его нужно открывать в браузере, чтобы запустить проект.
*   `/css/`
    *   `style.css` — Цветовая гамма, шрифты, стили кнопок (взяты со скриншота: темный фон, неоновый зеленый/желтый текст).
    *   `layout.css` — Стилизация окон для библиотеки Golden Layout.
*   `/js/`
    *   `main.js` — Отвечает за логику 3-х окон (Слева материал, по центру IDE, справа Задания).
    *   `editor.js` — Подключение и настройка Monaco Editor (движок от VS Code для подсветки синтаксиса).
    *   `compiler.js` — API-запросы к Piston или Judge0 для выполнения Java-кода.
    *   `roadmap.js` — Скрипт, который будет загружать данные из JSON и сохранять прогресс в localStorage.
*   `/data/`
    *   `roadmap.json` — **ВАЖНЫЙ ФАЙЛ**. Здесь должны лежать темы из твоего PDF файла!

## ⚙️ Инструкция по работе с `roadmap.json`

Поскольку у меня сейчас нет возможности скопировать текст из твоего PDF файла, тебе нужно будет перенести свой текст в файл `data/roadmap.json` самостоятельно по шаблону.

**Шаблон (структура одного урока):**
```json
{
    "id": "unik_name",
    "title": "Название темы из твоего PDF",
    "content": "Текст теоретического материала...",
    "task": {
        "description": "Текст практической задачи (например: Вывести 'Привет')",
        "expectedOutput": "Привет"
    }
}
```

## 🚀 Как запустить:
1. Зайди в папку с проектом.
2. Просто нажми дважды на файл `index.html`. Сайт откроется в браузере.
3. Код уже можно писать в редакторе по центру и нажимать "Run Code".
